# 📺 유튜브 사이트 목업 만들기 미션

구름 풀스택 20기 4팀 미니 프로젝트: 유튜브 목업 페이지 레이아웃 구현 과제입니다.

## 1. 프로젝트 개요
* **목표**: 유튜브 메인 페이지의 핵심 UI를 HTML/CSS만으로 구현하여 웹 퍼블리싱 기초 역량 습득
* **중점 사항**: 시멘틱 태그 활용, Flex/Grid 레이아웃 설계, 반응형 UI 구조 이해

## 2. 웹 서비스 목업 기획 및 설계
### UI/UX 디자인 분석
* **레이아웃**: 상단 고정 헤더(Header), 좌측 사이드바(Sidebar), 중앙 비디오 리스트(Main Content)의 3단 구조 설계
* **컬러 시스템**: 유튜브 다크 모드 테마 적용 (Background: `#0f0f0f`, Text: `#ffffff`)
* **컴포넌트**: 비디오 썸네일, 채널 프로필 이미지, 제목 및 조회수 정보가 포함된 카드형 UI 디자인

### 개발 문서 (Technical Specs)
* **HTML5**: 웹 접근성과 SEO를 고려하여 `header`, `nav`, `main`, `section` 등 시멘틱 태그 사용
* **CSS3**: 
  * `Flexbox`를 활용한 헤더 아이콘 및 검색창 정렬
  * `CSS Grid`를 활용한 비디오 카드 리스트의 규칙적인 배치
  * `Transition` 효과를 통한 버튼 및 썸네일 상호작용 구현
* **Assets**: Font Awesome 아이콘 및 유튜브 공식 로고 에셋 활용

## 3. 습득 역량 및 결과
* **HTML 마크업 기술**: 문서의 구조를 논리적으로 설계하는 능력 함양
* **CSS 스타일링 기술**: 정교한 수치 조정을 통한 UI 클론 구현 능력 습득
* **와이어프레임 기반 디자인**: 기획된 와이어프레임을 실제 웹 브라우저 렌더링 결과물로 변환
* **브라우저 렌더링 원리 파악**: HTML/CSS 파일 구조와 경로 설정이 렌더링에 미치는 영향 이해
